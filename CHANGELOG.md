## [0.1.0] - 2026-04-14

### Added

- Initial release of traverse-core component library
- Monaco KQL editor with syntax highlighting and contextual autocomplete
- Multi-cluster connection management with per-tab binding
- Virtual-scrolled results table (100K+ rows) with sort, filter, aggregates
- Auto-detected Vega-Lite charts (6 chart types)
- Schema browser with search, context menus, drag-and-drop
- Query history with IndexedDB-backed result recall
- Dark/light theme with system preference detection
- `bootstrapExplorer()` for standalone apps with MSAL auth
- traverse-web standalone SPA
- traverse-desktop Tauri wrapper
- VitePress documentation site
